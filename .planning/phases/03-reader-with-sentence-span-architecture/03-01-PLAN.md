  <behavior>
    ## Sentence model (D-03)
    - `Sentence('Hello.')` creates a Sentence with text 'Hello.'
    - Two Sentences with the same text are == equal
    - Two Sentences with the same text have the same hashCode
    - `Sentence('Hello.').toString()` returns 'Sentence(Hello.)'
    - `const Sentence('x')` compiles (const constructor)

    ## SentenceSplitter.split() (D-02)
    ### Basic splitting
    - `'Hello. World.'` => `[Sentence('Hello. '), Sentence('World.')]`
    - `'Hello! World?'` => `[Sentence('Hello! '), Sentence('World?')]`
    - `'Single sentence.'` => `[Sentence('Single sentence.')]`
    - `''` (empty string) => `[]`
    - `'No trailing punctuation'` => `[Sentence('No trailing punctuation')]`

    ### Abbreviations preserved (not split)
    - `'Mr. Smith went home. He rested.'` => 2 sentences (not 3)
    - `'Dr. Jones and Mrs. Smith met.'` => 1 sentence
    - `'St. Louis is nice. I agree.'` => 2 sentences
    - `'The U.S. is large. Very large.'` => 2 sentences

    ### Decimal numbers preserved
    - `'It cost $3.50 total.'` => 1 sentence
    - `'Pi is 3.14 approximately.'` => 1 sentence

    ### Ellipsis preserved
    - `'He waited... then left.'` => 1 sentence
    - `'Really?... Yes.'` => 2 sentences (split on ? not ...)

    ### Trailing quotes included with sentence
    - `'"Hello," she said. "Goodbye."'` => 2 sentences
    - `'She asked, "Why?" He shrugged.'` => 2 sentences

    ### Whitespace handling
    - Leading/trailing whitespace in input is trimmed
    - Whitespace between sentences is attached to the preceding sentence (trailing space) or preserved naturally
    - Multiple spaces between sentences collapse to single space attached to preceding sentence
  </behavior>

    ## Sentence model (lib/core/text/sentence.dart)
    Simple class with:
    - `final String text` field
    - `const` constructor: `const Sentence(this.text)`
    - `operator ==` comparing `text` field

    ## SentenceSplitter (lib/core/text/sentence_splitter.dart)
    Class with a single public method `List<Sentence> split(String text)`.

    Static const `_abbreviations` set containing: Mr, Mrs, Ms, Dr, Prof, Rev, Sr, Jr, St, Ave, Blvd, Gen, Gov, Sgt, Cpl, Pvt, Capt, Lt, Col, Maj, vs, etc, approx, dept, est, vol, Jan, Feb, Mar, Apr, Jun, Jul, Aug, Sep, Oct, Nov, Dec.

    Implementation approach -- character-by-character scan:
    1. If input is empty or whitespace-only, return `[]`
    2. Trim input
    3. Walk characters with index tracking
    4. When encountering `.`, `!`, or `?`:
       a. For `.`: check if preceded by a known abbreviation (lookbehind the word before the dot). If yes, skip. Check if preceded by a digit and followed by a digit (decimal number). If yes, skip. Check if followed by `.` (ellipsis). If yes, skip until end of dot sequence.
       b. For `!` and `?`: always a sentence boundary
       c. After the terminal punctuation, consume any closing quotes (`"`, `'`, `)`) that immediately follow
       d. Consume any trailing whitespace
       e. Emit the substring as a Sentence, start a new accumulator
    5. If text remains after the last boundary, emit it as a final Sentence

    The abbreviation check: extract the word immediately before the `.` by scanning backward from the dot position to the previous space or start-of-string. Check if that word (case-sensitive for capitalized abbreviations, case-insensitive for lowercase ones like `vs`, `etc`) is in `_abbreviations`.

    Single-letter uppercase followed by `.` (initials like `J. K. Rowling`) should NOT trigger a split -- check if the word before the dot is exactly 1 uppercase letter.
  </implementation>
